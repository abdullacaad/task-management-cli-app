## 🌐 Overview

🔗 **Repository:**
[https://github.com/abdullacaad/task-management-cli-app](https://github.com/abdullacaad/task-management-cli-app)

The **Task Management CLI App** is a lightweight Python-based command-line tool that helps users efficiently manage daily tasks directly from the terminal.

> 💡 Built for developers and students who prefer speed, simplicity, and keyboard-driven workflows.

---

## ✨ Key Features

* ✅ **Add Tasks** — Quickly create new tasks
* 📋 **View Tasks** — List all current tasks
* ❌ **Delete Tasks** — Remove completed or unwanted tasks
* ✔️ **Mark Complete** — Track progress easily
* ⚡ **Fast CLI Interface** — No GUI, pure speed

---

## 🏗 System Architecture

```
        User Input (CLI)
                │
                ▼
        Python Application
                │
        ┌───────┼────────┐
        ▼       ▼        ▼
     Add     Delete    Display
     Task     Task      Tasks
                │
                ▼
          Local Storage
```

> 🧠 The system processes commands and updates tasks instantly.

## ⚙️ How It Works

```text
1. User enters command in terminal
2. Python script parses input
3. Task data is updated accordingly
4. Results are displayed instantly
```

## 🚀 Getting Started

### 🔧 Prerequisites

* Python installed
* Terminal / Command Prompt

### 📦 Installation

```bash
git clone https://github.com/abdullacaad/task-management-cli-app
cd task-management-cli-app
```

### ▶️ Run the Application

```bash
python main.py
```

---

## 📸 Example Usage

```bash
# Add a task
python main.py add "Complete assignment"

# View tasks
python main.py list

# Mark complete
python main.py done 1

# Delete task
python main.py delete 1
```
